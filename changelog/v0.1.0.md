# v0.1.0 — 2026-04-19

Initial public release.

## Added

- Sidebar browser of all sessions under `~/.claude/projects/`, grouped by project and sorted by most-recent activity.
- Fuzzy filter across project paths and session titles (`Ctrl+K`). `Enter` launches the first visible session.
- One-click resume — each session opens in its own tab running `claude --resume <id>` inside an embedded VTE terminal.
- `+ New session` row per project for starting a fresh `claude` in that cwd.
- Embedded WebKit browser pane per tab (`Ctrl+Shift+B`), with back/forward/reload and URL bar.
- `--open-url URL` flag routes `xdg-open` / `$BROWSER` requests from inside `claude` into the current tab's browser pane.
- Live session-state dots in the sidebar: 🟢 working, 🟡 waiting, ⚪ idle.
- "Session ended" placeholder with **Resume** and **Close** buttons when `claude` exits.
- Nord-themed VTE terminal (JetBrains Mono, 8 px padding) and libadwaita light/dark theming.
- AUR packaging as `persistent-claude-code-git` plus pacman `.pkg.tar.zst` attached to each GitHub Release.
- Global keyboard shortcuts and About dialog.
- JSON config at `~/.config/persistent-claude-code/config.json` with corruption recovery.

## CI

- GitHub Actions `Release` workflow builds the pacman package inside an `archlinux:latest` container on every `v*.*.*` tag push and attaches the artefact to the Release (`contents: write` permission).
